# ev-assignment
Evolent assignment

## Getting Started

#### 1. Install gulp globally:

```sh
$ npm install --global gulp
```

#### 2. Install UI project Dependencies:

```sh
$ cd client
$ npm install
```

#### 3. Run gulp:

```sh
$ gulp
```
#### 4. Watch changes:

```sh
$ gulp watch
```

#### 5. Install server project devDependencies:

```sh
$ cd server
$ npm install
```

#### 6. Run app:

```sh
$ node app
```
